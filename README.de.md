# AWS EKS Terraform Observability Platform

DevOps- und Cloud-Projekt zur Bereitstellung einer Kubernetes-Infrastruktur auf AWS mit Terraform, Deployment einer containerisierten Anwendung auf Amazon EKS und Integration von Observability mit Prometheus und Grafana.

## Tech Stack

- AWS
- Terraform
- Amazon EKS
- Kubernetes
- Docker
- Prometheus
- Grafana
- Helm
- GitHub Actions
