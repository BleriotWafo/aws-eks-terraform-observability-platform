# AWS EKS Terraform Observability Platform

Projet DevOps et Cloud permettant de provisionner une infrastructure Kubernetes sur AWS avec Terraform, de déployer une application conteneurisée sur Amazon EKS et d’ajouter une couche d’observabilité avec Prometheus et Grafana.

## Stack technique

- AWS
- Terraform
- Amazon EKS
- Kubernetes
- Docker
- Prometheus
- Grafana
- Helm
- GitHub Actions
